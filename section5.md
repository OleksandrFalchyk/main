# РОЗДІЛ 5. Відображення User Stories у UML-нотації

Для візуалізації функціональних можливостей системи використано UML-діаграму варіантів використання (Use Case Diagram).

---

## UML-діаграма (PlantUML)

```plantuml
@startuml
left to right direction

actor Адміністратор
actor "Працівник деканату" as Dekanat
actor Викладач
actor Студент

rectangle "ІС обліку студентів" {

  Адміністратор -- (Створення користувача)
  Адміністратор -- (Призначення ролі)
  Адміністратор -- (Блокування користувача)

  Dekanat -- (Додавання студента)
  Dekanat -- (Редагування даних студента)
  Dekanat -- (Формування списків груп)
  Dekanat -- (Перегляд статистики)

  Викладач -- (Перегляд списку групи)
  Викладач -- (Виставлення оцінок)
  Викладач -- (Редагування оцінок)

  Студент -- (Перегляд власних оцінок)
  Студент -- (Перегляд інформації про групу)
}

@enduml
